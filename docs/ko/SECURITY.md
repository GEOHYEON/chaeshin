# Security Policy

## 보안 고려사항

Chaeshin은 LLM 출력을 기반으로 도구를 실행하는 프레임워크입니다.
다음 보안 사항을 반드시 인지하고 사용하세요.

### LLM 출력 신뢰성

- LLM이 생성한 그래프(replan)가 유효한지 **반드시 검증**하세요
- `max_loops` 설정으로 무한 루프를 방지합니다 (기본값: 3)
- LLM이 존재하지 않는 도구를 호출하면 `GraphExecutor`가 자동 차단합니다

### API 키 관리

- `.env` 파일에 API 키를 저장하세요
- `.env` 파일을 **절대** Git에 커밋하지 마세요
- `.env.example`을 참고하여 필요한 환경변수를 설정하세요

### 도구 실행 격리

- 도구의 `executor` 함수에서 시스템 명령어 실행을 **제한**하세요
- 사용자 입력을 도구 파라미터로 전달할 때 **검증**하세요
- 프로덕션 환경에서는 도구 실행을 샌드박스에서 수행하는 것을 권장합니다

### 케이스 저장소

- CBR 케이스에 민감한 개인정보가 포함되지 않도록 주의하세요
- 의료, 금융 등 민감한 도메인에서는 케이스 익명화를 적용하세요
- VectorDB 접근 권한을 적절히 관리하세요
